# 🌡️ Temperature, Humidity & Pressure Monitoring System

## 📌 Description
This project is a **Weather Monitoring System** using Arduino that measures **Temperature, Humidity, and Atmospheric Pressure**. It uses sensors like **DHT11/DHT22** for temperature & humidity and **BMP180/BMP280** for pressure. The data can be displayed on an LCD/OLED or Serial Monitor.

---

## 🔧 Features
- 🌡️ Measures Temperature  
- 💧 Measures Humidity  
- 🌬️ Measures Atmospheric Pressure  
- 📟 Displays data on Serial Monitor / LCD / OLED  
- ⚡ Real-time monitoring  

---

## 🧰 Components Used
- Arduino Uno  
- DHT11 or DHT22 Sensor  
- BMP180 or BMP280 Sensor  
- LCD 16x2 / OLED Display (optional)  
- Breadboard  
- Jumper Wires  
- Resistors  

---

## ⚙️ Working Principle
1. The **DHT sensor** reads temperature and humidity  
2. The **BMP sensor** reads atmospheric pressure  
3. Arduino processes the data  
4. Values are displayed on Serial Monitor or LCD/OLED
